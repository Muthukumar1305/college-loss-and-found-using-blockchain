The Lost and Found for College Campus using Blockchain is a decentralized web application designed to help students report, track, and recover lost items securely. By leveraging blockchain technology, the system ensures transparency, tamper-proof records, and trust among users.

🎯 Objectives
Provide a secure platform for reporting lost and found items
Ensure data integrity using blockchain
Reduce fraud and false claims
Enable easy tracking of items
🚀 Features
🔐 Blockchain-based storage (immutable records)
👤 User authentication system
📤 Report lost items
📥 Report found items
🔍 Search & filter items
🔗 Unique tracking ID for each item
📊 Transparent transaction history
🏗️ System Architecture

The system consists of:

Frontend
HTML, CSS, JavaScript (or React)
Backend
Node.js / Express (optional)
Blockchain Layer
Smart contracts for storing item details
Ethereum / local blockchain (like Ganache)
Database (Optional)
MongoDB for user data
⚙️ Tech Stack
Frontend: HTML, CSS, JavaScript / React
Backend: Node.js, Express
Blockchain: Ethereum, Solidity
Tools: Ganache, MetaMask
Database: MongoDB (optional)
